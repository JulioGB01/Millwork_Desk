# Contributing

Thank you for improving Millwork Desk.

## Good contributions

Report an incorrect definition, broken interaction, accessibility issue, or unclear reference. Keep pull requests focused and explain the user-facing result.

## Standards material

Do not submit copyrighted AWS/AWI/AWMAC/WI text, images, or other licensed material unless you have permission to do so. Contributions should be concise, original reference guidance and must not present this project as an official standards publication.

## Before opening a pull request

Test the interface in a modern browser, preserve keyboard accessibility, and update the README whenever the public scope or deployment process changes.
